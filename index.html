Dépannage

Troubleshooting

Les problèmes les plus courants et leurs solutions

Common issues and their solutions

🐍 Python n'est pas reconnuPython is not recognized

Ouvre un terminal (cmd) et tape python --version. Si "n'est pas reconnu" s'affiche, réinstalle Python et coche "Add Python to PATH".

Open a terminal (cmd) and type python --version. If "not recognized" appears, reinstall Python and check "Add Python to PATH".

🍪 Les cookies ne fonctionnent pasCookies don't work

Assure-toi d'être connecté à YouTube dans le navigateur. Ouvre YouTube.com, vérifie que ton avatar apparaît en haut à droite, puis clique sur Sync cookies dans l'extension.

Make sure you're signed in to YouTube. Open YouTube.com, check that your avatar appears top-right, then click Sync cookies in the extension.

🔌 Le port 8765 est déjà utiliséPort 8765 is already in use

Un autre programme utilise le port. Tu peux :

  • Lancer python reset.py pour tuer tous les processus et réinitialiser
  • Changer le port dans backend/.env (variable NASTUBE_PORT)

Another program is using the port. You can:

  • Run python reset.py to kill all processes and reset
  • Change the port in backend/.env (variable NASTUBE_PORT)

📥 Les téléchargements échouentDownloads are failing

Causes possibles :

  • Cookies expirés → re-synchronise (étape 5)
  • YouTube a changé → mets à jour yt-dlp : .venv\Scripts\pip install -U yt-dlp
  • Plus d'espace disque → libère de l'espace ou change le disque de stockage

Possible causes:

  • Expired cookies → re-sync (step 5)
  • YouTube changed → update yt-dlp: .venv\Scripts\pip install -U yt-dlp
  • Disk full → free up space or change storage drive

🧹 Tout réinitialiser (reset)Reset everything

Ouvre un terminal dans le dossier NASTube/ et tape python reset.py. Cela supprime :

  • Tous les processus liés à NASTube
  • Les vidéos et la base de données
  • L'environnement virtuel Python (.venv)
  • La configuration (.env), les scripts, les raccourcis
  • Les dossiers backend/, extension/, frontend/ sont conservés

Open a terminal in the NASTube/ folder and run python reset.py. This removes:

  • All NASTube-related processes
  • Videos and database
  • Python virtual environment (.venv)
  • Configuration (.env), scripts, shortcuts
  • The backend/, extension/, frontend/ folders are kept
